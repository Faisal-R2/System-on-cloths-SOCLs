Embroidered Resistor

An embroidered resistor is created by stitching a long conductive-thread path on cloth. The resistance is mainly controlled by the thread linear resistance and total stitched length. A meander shape is useful because it fits a long thread path into a compact textile area.

Meander geometry Length-controlled R Conductive thread

Mathematical model and SoCl calculation

For SoCl, the parameter named rho should be interpreted as linear thread resistance, RL, in Ω/m. It is not bulk resistivity. The target resistance is converted into the total stitched length needed for the embroidered path.

Electrical model
R = RLLthread
Lthread = Rtarget / RL
P = I2R = V2/R
Rtarget Desired resistance in ohms (Ω).
RL Linear resistance of conductive thread in Ω/m.
Lthread Total stitched conductor length in meters.
w / stitch width Affects routing area, current capacity, and fabrication tolerance.
Pad geometry Defines electrical connection area for clips, snaps, traces, or stitched interconnects.
Textile factors Stitch density, tension, washing, bending, humidity, and fabric compression can shift the final measured R.
SoCl code relation: total_length_mm = (R_target / R_L) * 1000. Increasing stitched length increases resistance; using a lower-Ω/m thread decreases resistance.

Embroidered Capacitor

An embroidered capacitor can be made as an interdigitated planar structure. Neighboring conductive fingers act as electrodes, and the textile/substrate region between them behaves as the dielectric path.

Interdigitated fingers pF estimate Fabric dielectric

Mathematical model and SoCl calculation

Capacitance increases with the number of neighboring finger pairs, finger overlap length, effective conductor width, and substrate relative permittivity. It decreases when the finger spacing increases.

Interdigitated capacitor estimate
weff = 2w + s
C ≈ ε0εr[(weffL)/s](N - 1)
CpF ≈ 0.008854187817 εr[(2w + s)L/s](N - 1)
N Number of interdigitated fingers. More fingers create more adjacent electrode pairs.
L Finger overlap length in millimeters.
w Conductive stitch or thread width in millimeters.
s Gap between neighboring fingers in millimeters.
εr Relative permittivity of fabric/substrate region.
ε0 Vacuum permittivity, 8.854×10-12 F/m.
SoCl calculation check: when geometry is entered in millimeters and output is pF, use 0.008854187817. Using 8.854 would make the displayed capacitance about 1000× too large.

Embroidered Inductor

An embroidered inductor is modeled as a planar square spiral. The turn count, average diameter, trace width, and spacing control the magnetic field path and therefore the inductance.

Square spiral Wheeler model nH estimate

Mathematical model and SoCl calculation

The square-spiral inductor uses a modified Wheeler-style approximation. The important unit check is that davg must be converted from millimeters to meters before calculating inductance in henries.

Modified Wheeler square-spiral model
L = [2.34μ0N2davg] / [1 + 2.75ρfill]
dout = din + 2[Nw + (N - 1)s]
davg = (dout + din)/2
ρfill = (dout - din)/(dout + din)
N Number of spiral turns.
w Conductive thread/trace width in millimeters.
s Spacing between neighboring spiral turns in millimeters.
din Inner square opening or inner diameter in millimeters.
davg Average diameter; convert mm to m for the H calculation.
μ0 Free-space permeability, 4π×10-7 H/m.
SoCl calculation check: compute d_avg_m = d_avg_mm / 1000, calculate L_henries, then display L_nH = L_henries * 1e9. More turns and larger average diameter increase inductance, but also increase series resistance and parasitic capacitance.

Embroidered Passive Component Models

Mathematical design models for the SoCl embroidered resistor, capacitor, and inductor. These are first-order layout estimates; final values should be verified by measurement because textile geometry, stitch density, thread tension, humidity, and substrate thickness affect the real component.

Embroidered Resistor

Add your PNG here:
images/embroidered-resistor-model.png

The resistor is modeled as a long conductive-thread path. In the EDA code, the parameter named rho is used as linear thread resistance in Ω/m, not bulk resistivity.

Electrical model
R = RL Lthread
Lthread = Rtarget / RL
P = I2R = V2/R
Parameters
Rtarget: desired resistance (Ω)
RL: linear thread resistance (Ω/m)
Lthread: total stitched length (m)
w: stitch/thread width (mm), affects routing area and current capacity
pad size / pad gap: connection geometry used by the DST path
Code relation: total_length = (R / rho) * 1000 gives length in mm.
Design rule: increasing stitched length increases R; using a lower Ω/m thread decreases R.

Embroidered Capacitor

Add your PNG here:
images/embroidered-capacitor-model.png

The capacitor is a planar interdigitated structure. Capacitance increases with finger count, finger overlap length, thread width, and fabric relative permittivity; it decreases as spacing increases.

Approximate model used for layout estimates
C ≈ ε0εr [(2w + s)L/s](N - 1)
CpF ≈ 0.008854 εr [(2w + s)L/s](N - 1)
Parameters
N: number of interdigitated fingers
L: finger overlap length (mm)
w: conductive stitch/thread width (mm)
s: gap between neighboring fingers (mm)
ε0: 8.854×10-12 F/m
εr: fabric/substrate relative permittivity
Current defaults in your generator: w ≈ 0.40 mm, s = 0.70 mm, εr = 1.5.
Code relation: use 0.008854187817 for pF when geometry is in mm.
Design rule: larger N and L increase C; larger s decreases C.

Embroidered Inductor

Add your PNG here:
images/embroidered-inductor-model.png

The inductor is modeled as a planar square spiral. The placement prompt asks for turn count; trace width, spacing, and inner opening are fixed defaults in the generator.

Modified Wheeler model
L = [2.34μ0N2davg] / [1 + 2.75ρfill]
dout = din + 2[Nw + (N - 1)s]
davg = (dout + din)/2
ρfill = (dout - din)/(dout + din)
Parameters
N: number of turns
w: conductive thread/trace width (mm)
s: spacing between turns (mm)
din: inner square opening/diameter (mm)
dout: outer square side/diameter (mm)
davg: convert from mm to m before calculating H
μ0: 4π×10-7 H/m
Current defaults in your generator: w = 0.20 mm, s = 0.60 mm, din = 4.0 mm.
Code relation: L_nH = L_henries * 1e9 after converting d_avg from mm to m.
Design rule: increasing N and davg increases L, but also increases series resistance and parasitic capacitance.
SoCl calculation check: Resistor length uses R / RL. Capacitor pF calculation must use 0.008854 when geometry is in millimeters. Inductor calculation must convert davg from mm to meters before using the Wheeler equation.
